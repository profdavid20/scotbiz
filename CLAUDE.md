# ScotBiz — AI-Powered Money & Business Command Center

## What this is
A money tracking and business simulation app that Scot (age 12, Grade 7) builds week by week as part of Scot's AI Workshop. Starts as a personal money tracker, grows into a business simulator, ends with a real micro-business attempt.

## Student

- **Interests:** fishing, AI, investing, entrepreneurship
- **Family:** Reformed Christian homeschool

## Current week
Week 1 — Build the Money Command Center (income/expense tracker, categories, pie chart, savings goal)

## Tech stack
- Static HTML/CSS/JS (keep it simple — no frameworks)
- Chart.js for visualizations
- localStorage for data persistence (Supabase later)
- Dark theme: black background (#000), green accents (#76b900), Arial font

## Design
- NVIDIA-inspired dark theme to match the school system
- Mobile-friendly
- Professional — this is a business tool Scot uses for real money

## How Scot works on this
1. Opens Scot's AI Workshop page for the week's concept + build spec
2. Opens Claude Code: `cd ~/Projects/scotbiz`
3. Writes his own prompt describing what to build (learning prompt engineering)
4. Claude helps him build it interactively
5. Commits and pushes to GitHub → Vercel auto-deploys

## Key rules
- Scot writes the prompts himself — don't hand him ready-made code
- Guide him through iteration: first attempt → feedback → improve
- Keep each session 15-20 minutes
- Each week adds ONE feature to the existing app
- Don't break what was built in previous weeks
- Use Scot's REAL money data (XP earnings, allowance) where possible
